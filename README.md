Here's your complete project:
Code (firewall_simulator):
firewall.py — core engine: Rule, Packet, Firewall classes with priority-based rule matching (supports CIDR subnets and port ranges), default-deny policy, and a sliding-window rate limiter for DoS/port-scan detection
packet_generator.py — generates simulated normal traffic plus attack patterns (port scan bursts, DoS floods)
main.py — interactive CLI menu to add/view rules, run simulations, launch attack scenarios, check stats, view blocked IPs, and export logs
rules.json — sample rule set (allow HTTP/HTTPS, restrict SSH to LAN, block Telnet/RDP, blacklist an IP)
Run it with python3 main.py from that folder — no external dependencies needed.
